# Создание репозитория
```sh
git init
```
## Добавить файл к отслеживанию
```sh
git add
```
## Добавить коммит с сообщением
```sh
git commit -m"message"
```
## Список всех коммитов
```sh
git log
```
### Список коммитов упрощённый сокращенный
```sh
git log --oneline
```
## Команда для переключения между коммитами
```sh
git checkout
```
